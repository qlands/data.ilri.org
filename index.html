Data Portal

The data portal makes ILRI's open access datasets globally available. The platform is also the main access point for data from the CGIAR Research Program on Livestock and Fish (CRP) and from other CGIAR research programs and centres as part of our cross-centre collaboration.

Data portal »

Tools

The tools portal provides metadata and documentation for a wide range of research tools used by the CGIAR and our partners. The portal also enables interaction among users and offers a platform for providing feedback on adaptation and use of the tools.

Tools portal »

Code

Source code for a handful of ILRI projects is hosted on GitHub, promoting an open-by-default workflow and enabling community interaction. The code includes systems written in house as well as community-maintained open-source projects we use.

GitHub »

GIS Portal

The ILRI GIS portal provides access to web-based geospatial information and services. The portal contains both ILRI, hosted and publicly available spatial datasets that are discoverable, viewable and accessible by others.

GIS portal »

Outputs

CGSpace is a repository of agricultural research outputs and results produced by different parts of CGIAR and partners. It indexes reports, articles, press releases, presentations, videos, policy briefs and more.
